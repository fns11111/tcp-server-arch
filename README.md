## Вариант задания

### Задание 1: TCP Server на Arch Linux
- Базовый образ: archlinux:latest
- Язык: C++, Qt6
- Открыть порт 1234
- Сервер принимает подключения и отправляет приветствия
- Docker образ на Docker Hub: aasofroneev/tcp-server-arch

### Задание 2: Multi-Client Chat Server
- Поддержка до 5 одновременных клиентов
- При подключении сервер запрашивает ник
- При переполнении отправляет сообщение и разрывает соединение
- Команды: /nick, /quit, /clients
- Логирование всех событий в chat_log.txt
- Docker образ на Docker Hub: aasofromeev/chat-server
